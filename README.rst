================
Dozenal Calendar
================

This is a package to make working with a dozenal calendar of my own devising
easier.

------------
Installation
------------

Create a virtual environment and then run setup.py::

    $ python3 -m venv venv
    (venv)$ source venv/bin/activate
    (venv)$ python setup.py install

-------
Testing
-------

Simply run py.test::

    (venv)$ py.test
